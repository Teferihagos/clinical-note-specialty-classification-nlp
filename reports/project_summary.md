# Project Summary

This project built an end-to-end NLP pipeline to classify clinical note text into medical specialties using the MTSamples dataset.

The workflow included data cleaning, text preprocessing, TF-IDF vectorization, model training, and evaluation. Two models were tested: Logistic Regression and Linear SVM.

Logistic Regression performed better, achieving 38.5% accuracy compared to 27.5% for Linear SVM. The results showed that class imbalance had a strong effect on performance, with dominant classes such as Surgery receiving better predictions than smaller specialties.

This project demonstrates practical healthcare NLP skills, including text preprocessing, feature engineering, multiclass classification, model comparison, and interpretation of confusion matrices.
